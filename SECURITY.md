# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| latest release | :white_check_mark: |
| older releases | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Medio, please report it responsibly:

1. **Do NOT open a public GitHub Issue.**
2. Email the maintainer directly (see the repo owner's GitHub profile for contact info).
3. Include a clear description of the vulnerability, steps to reproduce, and potential impact.

If the vulnerability is confirmed, a fix will be prioritized and a security advisory published.

## Scope

This policy covers the Medio client application only. Vulnerabilities in third-party dependencies should be reported to their respective maintainers.
